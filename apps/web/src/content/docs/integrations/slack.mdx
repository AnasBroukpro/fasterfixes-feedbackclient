---
title: Slack integration setup
description: Set up the Faster Fixes Slack integration to get notified in a Slack channel when feedback arrives or changes.
---

Faster Fixes integrates with Slack to notify you in a channel of your choice when feedback is created or its status changes. This is a one-way notification channel — it does not create issues in Slack; for two-way issue tracking, use the [GitHub](/docs/integrations/github) or [Linear](/docs/integrations/linear) integrations instead. For a product overview, see the [Slack integration overview](/integrations/slack).

## Setup

1. Go to **[Application Settings](/integrations) → Integrations** from the sidebar
2. Click **Connect Slack** and authorize the Faster Fixes app on your workspace
3. In **[Project Settings](/settings) → Slack**, pick the channel that should receive feedback
4. Leave posting enabled, or toggle it off without disconnecting

The bot is installed once per organization. Each project then links to a single channel within that workspace.

## Notifications

Faster Fixes notifies the linked channel when feedback is created or its status changes. The full diagnostic trail (DOM selector, browser info, console logs, network requests) stays in the dashboard — follow the link in the message to see it.

## Channels

Public channels are selectable from a dropdown out of the box. Private channels are supported only if the Faster Fixes bot is invited to them first.

## One channel per project

Each project links to its own channel, so feedback from different client projects stays isolated rather than landing in a single shared firehose. You can change a project's channel or turn posting off at any time without affecting other projects.

## Disabling and disconnecting

You can turn posting off per project from **[Project Settings](/settings) → Slack** without disconnecting. Disconnecting from **[Application Settings](/integrations) → Integrations** removes the workspace install and cascades all project links. Existing Slack messages are not deleted; they simply stop updating.
